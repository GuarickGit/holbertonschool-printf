					 _printf Manual Page
------------------------------------------------------------

NAME
	_printf - produce output according to a format string

SYNOPSIS
	#include "main.h"

	int _printf(const char *format, ...);

DESCRIPTION
	The _printf() function produces output according to a format string
	and a variable number of arguments.

	Supported conversion specifiers:
		%c  - print a single character
		%s  - print a string
		%%  - print a percent sign
		%d  - print a signed decimal integer
		%i  - print a signed decimal integer

RETURN VALUE
	This function returns the number of characters printed (excluding the
	null byte used to end output to strings).

NOTES
	- Does not handle flags, field width, precision or length modifiers.
	- Writes directly to standard output using the _putchar function.
	- Unsupported specifiers are printed as-is (e.g., %r → %r).

EXAMPLES
	_printf("Character: %c\n", 'H');
	_printf("String: %s\n", "Holberton");
	_printf("Integer: %d\n", 53);
	_printf("Percent: %%\n");
